Core Features

Everything You Need in One File

No backend, no dependencies, no login required. Everything runs in the browser and persists via localStorage.

Reference Library

Meeting frameworks, discovery question banks, QBR checklists, feature request guides, and de-escalation playbooks — all the CS knowledge you need in structured, searchable sections.

Interactive Tools

Weekly RFS planner, client health tracker with Red/Yellow/Green status, and hero journey mapper — tools that store data locally and export to plain text.

Discovery System

Bob London's radically authentic discovery questions, vague-to-measurable outcome conversion guide, and call assessment scorecard with interactive sliders.

Risk Management

3-phase detection/categorization/mitigation system with weekly risk checklist and 11 toggleable risk category flags for portfolio tracking.

Renewal & Expansion

JRP development checklist (120-day), SPOON expansion framework, and structured renewal conversation guides with objection handling.

AI Integration

PLAN framework with 3 ready-to-use prompts for pre-meeting briefings, discovery question generation, and call assessment — designed for Claude and ChatGPT.
